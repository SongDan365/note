```vim
" 文件中插入系统时间
a<C-R>=strftime("%c")<CR><esc>

'.    移动光标到上一次修改的行
`.    它移动光标到上一次修改点
.     重复上次命令
<c-o>    依次沿着你的跳转记录向回跳 (从最近的一次开始)
<c-i>    依次沿着你的跳转记录向前跳
:ju(mps)    列出你跳转的足迹
:history    列出历史命令记录
:his c    命令行命令历史
:his s    搜索命令历史
g <c-g>    计算文件字符
{    光标移动到上一段落
}    光标移动到下一段落
gg    光标移动到第一行第一个字符
G    光标移动到最后一行第一个字符
ci[    删除[]中的所有字符并进入插入模式
ci(    删除()中的所有字符并进入插入模式
ci<    删除<>中的所有字符并进入插入模式
ci{    删除{}中的所有字符并进入插入模式
cit    删除HTML/XML的标签内部的所有字符并进入插入模式
ci"    删除""中所有字符并进入插入模式
ci'    删除''中所有字符并进入插入模式
ci`    删除``中所有字符并进入插入模式
vi     同ci
di     同ci
```
